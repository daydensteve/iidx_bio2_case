Simple case to hold a BIO2 board for use with IIDX (BI2A). Includes 38mm hoels to allow passing wires into PC case via a grommet.

![Example Installation](doc/bio2case.jpg)

Note: The M3 standoffs are on the looser side - consider some grip strips or locktite to keep the board secure.

Example parts:
- PC Case: https://www.amazon.com/dp/B0F2HHXG5K
- Grommet: https://www.amazon.com/dp/B0DSJJDSGF
- x2-5 M4 bolts (to mount case to PC)
  - in my case, I only used 2 (front corners) + the grommet
- x4 M3 bolts (to mount BIO2 to case)
